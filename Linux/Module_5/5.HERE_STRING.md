Convert a string to uppercase using 1)HERE DOCUMENT 2)HERE STRING


            #!/usr/bin/env bash
            tr a-z A-Z << HERE
              welcome
              to
              bash
              learning
            HERE

            tr a-z A-Z <<< 'welcome to bash learning'
